# Stock-TimeFM React Dashboard

React + TypeScript + Vite dashboard for the Stock-TimeFM backend in
`~/stock-timefm`. The backend FastAPI service runs on `:8001`; Vite proxies
`/api`, `/health`, `/predict`, and `/ws` to it during dev and preview.

## Current Routes

| Route | Page |
|---|---|
| `/` | Command Center |
| `/command-center` | Alias for Command Center |
| `/pillars` | Pillars |
| `/traders` | Trader leaderboard and signals |
| `/map` | Bay Area company map |

## Commands

```bash
npm run dev        # Vite dev server on :3000
npm run typecheck  # strict TypeScript project build, no bundle
npm run build      # typecheck + production bundle
npm run lint       # ESLint source gate
npm run preview    # serve dist/ on :3000 with backend proxy
```

## Notes

- `dev-dist/` is generated by PWA/dev tooling and is ignored.
- PWA icons in `public/` are source assets because `vite.config.ts` references
  them directly.
- Active navigation is defined in `src/lib/constants.ts::NAV_ITEMS`; route
  components are wired in `src/App.tsx`.
- The deprecated forecast/eval/intel/signals standalone pages were removed.
